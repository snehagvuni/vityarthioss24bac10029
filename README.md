## Description of Scripts

### Script 1
Displays Linux system identity including kernel version, user name, uptime, distribution name, current date and license.

### Script 2
Checks whether Python package is installed and prints version details.

### Script 3
Audits important Linux directories for size, ownership and permissions.

### Script 4
Reads a log file and counts keyword occurrences.

### Script 5
Generates a personalised open source manifesto and saves output to text file.
